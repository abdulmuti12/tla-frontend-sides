# TLA-2024 - Luxury Asia Living Group Website

![Luxury Asia Living Group Website Preview](/public/preview-screenshot.png)

This is a corporate website for Luxury Asia
Living Group, a leading authority in luxury interior design and furnishings. The website is built
with [Next.js](https://nextjs.org/) and uses yarn as the package manager.

## About Luxury Asia Living Group

Luxury Asia Living Group represents renowned European brands like Gamma, Lago, Bonaldo, and Lamborghini, alongside
legacy brands Casa Italia, Polflex Office, Homelogy, Melba, and Masterpiece. Since 1995, they have delivered
unparalleled quality and design innovation to homes, offices, and hospitality projects across the globe.

## Features

- Brand showcase
- Featured projects gallery
- Press releases
- Product catalogues
- Client portfolio
- Contact information
- About us section

## Getting Started

First, install the dependencies:

```bash
yarn install
```

Then, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Project Structure

- `src/app` - Next.js app router pages
- `src/components` - React components organized by page/feature
- `src/api_helpers` - API integration functions
- `src/store` - State management using Zustand
- `src/types` - TypeScript type definitions
- `public/assets` - Static assets like images

## Technologies Used

- [Next.js](https://nextjs.org/) - React framework
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [Zustand](https://github.com/pmndrs/zustand) - State management
- [Framer Motion](https://www.framer.com/motion/) - Animations
